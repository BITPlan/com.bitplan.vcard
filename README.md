Maven dependency

```xml
<dependency>
  <groupId>com.bitplan.vcard</groupId>
  <artifactId>com.bitplan.vcard</artifactId>
  <version>0.0.1</version>
</dependency>
```

### How to build
```
git clone https://github.com/BITPlan/com.bitplan.vcard
cd com.bitplan.vcard
mvn install
```
